# 기능 목록
- 메세지 출력하기
- 사용자로부터 입력받기
- 구입 금액으로부터 몇개의 로또를 살지 계산하기
- 로또 번호 랜덤 생성
- 6개 번호로 구성된 1개의 로또 생성
- 당첨 번호 저장
- 보너스 번호 저장
- 로또 결과 반환
- 예외
  - 로또 구입 금액을 입력받는 경우 
    - 로또 구입 금액이 숫자가 아닌 경우
    - 로또 구입 금액 1000으로 나누어 떨어지지 않는 경우
  - 당첨 번호를 입력받는 경우
    - 한자리 숫자 6개 사이에 쉽표 1개의 형식이 지켜지지 않았을 경우
  - 보너스 번호를 입력받는 경우
    - 한자리 숫자 1개가 아닐 경우
